Rocket Rush - 2D Physics Space Game
A momentum-based arcade game developed in Unity 6 as a BCA Minor Project.

[Play the Game Here (Itch.io)](https://sid123456-alt.itch.io/rocketrush) 

Project Overview:-
Rocket Rush is a high-octane survival game where players navigate a rocket through an asteroid field. The project focuses on **realistic inertia** and **physics-based movement** to simulate a space environment.

Key Features:-

- Custom Physics: Implemented using AddForce and Rigidbody2D for authentic momentum.
- VFX & Particles: Dynamic thruster fire and explosion effects using Unity Particle Systems.
- Pixel Art: Hand-drawn assets and sprites for a classic indie aesthetic.
- Responsive UI: Modern HUD showing real-time score tracking.

 Technical Stack:-

- Engine: Unity 6 (Universal Render Pipeline)
- Scripting: C# (Object-Oriented Programming)
- Deployment: WebGL for browser-based playability.

Repository Structure:-

- Assets: Core C# scripts, Sprites, and Prefabs.
- ProjectSettings: Custom physics and input configuration.
